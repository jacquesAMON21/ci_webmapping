Ce projet de webmapping interactif vise à visualiser et explorer les localités (villes et villages) ainsi que les sous-préfectures de la Côte d’Ivoire à travers une interface cartographique intuitive et dynamique.

L’objectif est de faciliter :

la consultation des divisions administratives du pays,

la recherche de localités sur la carte,

et la visualisation des limites administratives à différents niveaux.

Ce projet constitue un support utile pour la planification territoriale, la gestion des données géographiques, ou encore l’analyse spatiale.


Les données proviennent de sources ouvertes et reconnues :

GADM (Global Administrative Areas)
→ Fournit les limites administratives officielles de la Côte d’Ivoire (districts, régions, départements, sous-préfectures).
* Source : https://gadm.org

Humanitarian Data Exchange (HDX / OCHA)
→ Fournit la base des localités (villages, villes, hameaux) avec leurs coordonnées géographiques.
* Source : https://data.humdata.org

NB: Les couches ont été harmonisées, nettoyées et intégrées dans une base de données spatiale avant publication.
